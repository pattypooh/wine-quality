API Documentation : wine quality prediction ❤️

Endpoints

This end point allows you to make a prediction of a wine quality, from 1 to 10, according to some parameters

How to request this endpoint

You should request this endpoint using a POST method with JSON content, Your JSON should contain the values, separated by comas, for the following parameters

  • Parameter 1: fixed acidity
  • Parameter 2: volatile acidity
  • Parameter 3: citric acid
  • Parameter 4: residual sugar
  • Parameter 5: chlorides
  • Parameter 6: free sulfur dioxide
  • Parameter 7: total sulfur dioxide
  • Parameter 8: density
  • Parameter 9: pH
  • Parameter 10: sulphates
  • Parameter 11: alcohol
Example using curl in localhost:
curl -i -H "Content-Type: application/json" -X POST -d '{"input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]]}' http://localhost:5000/predict

If you're using the deployed application:
curl -i -H "Content-Type: application/json" -X POST -d '{"input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]]}' https://pez-wine-app.herokuapp.com/predict

Example using requests in localhost:
import requests
response = requests.post("http://127.0.0.1:5000/predict", json={"input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]]})
response.json()


Example using requests with the deployed app:
import requests
response = requests.post("https://pez-wine-app.herokuapp.com/predict", json={"input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]]})
response.json()
